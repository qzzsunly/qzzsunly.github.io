9. System Requirements

By default, Spring Boot 2.0.0.BUILD-SNAPSHOT requires Java 7 and Spring Framework 5.0.0.BUILD-SNAPSHOT or above. You can use Spring Boot with Java 6 with some additional configuration. See Section 79.11, “How to use Java 6” for more details. Explicit build support is provided for Maven (3.2+) and Gradle 2 (2.9 or later). Gradle 3 is not supported.

Tip

Although you can use Spring Boot with Java 6 or 7, we generally recommend Java 8 if at all possible.
